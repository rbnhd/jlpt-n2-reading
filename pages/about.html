このサイトについて

JLPT N2読解練習サイトへようこそ

🎯 サイトの目的

このサイトは、JLPT（日本語能力試験）N2レベルの読解力を向上させたい学習者のために作られた無料の練習プラットフォームです。

実際の試験形式に基づいた練習問題を通じて、効率的に読解スキルを磨くことができます。

✨ 主な機能

📚 豊富な練習問題

JLPT N2の5つの問題タイプ（短文、中文、統合理解、主張理解、情報検索）をカバーする15以上の読解パッセージを用意しています。

📖 語彙・文法学習

1600以上のN2レベルの語彙と100の文法パターンを、カテゴリー別に学習できます。検索・フィルター機能で効率的に復習できます。

📊 進捗管理

解答履歴、正答率、カテゴリー別の成績、連続学習日数などを自動的に記録。あなたの成長を可視化します。

💯 即座のフィードバック

各問題の正解・不正解を即座に確認でき、解説を通じて理解を深めることができます。

📱 レスポンシブ対応

PC、タブレット、スマートフォンなど、どのデバイスからでも快適に学習できます。

🔒 プライバシー重視

すべてのデータはあなたのブラウザに保存され、サーバーには送信されません。安心して使えます。

📋 コンテンツの構成

読解パッセージ（15以上）

  • 短文： 250-350字程度の短い文章（5パッセージ）
  • 中文： 550-750字程度の中程度の文章（6パッセージ）
  • 統合理解： 2つの文章を比較（2パッセージ）
  • 主張理解： 900-1200字の論説文（3パッセージ）
  • 情報検索： 表やリストからの情報検索（2パッセージ）

語彙（1600語以上）

8つのカテゴリーに分類：

  • 社会・文化（200語）
  • 日常生活（200語）
  • ビジネス（200語）
  • 学術（200語）
  • 抽象概念（200語）
  • 日常活動（200語）
  • 抽象的概念（200語）
  • 技術（200語）

文法（100パターン）

N2レベルで必要な文法項目を網羅し、例文と解説を付けています。

🚀 使い方

  1. 練習問題を選ぶ

    「練習問題」ページから、問題タイプやカテゴリーを選んで練習を始めましょう。

  2. 問題を解く

    文章を読んで、選択肢から正解を選びます。時間を意識して解くと本番の練習になります。

  3. 結果を確認

    解答後、すぐに正解・不正解がわかります。解説を読んで理解を深めましょう。

  4. 進捗をチェック

    「進捗」ページで、自分の学習状況や成績を確認できます。

  5. 語彙・文法を復習

    わからない単語や文法があれば、「語彙」「文法」ページで復習しましょう。

🛠️ 技術仕様

このサイトは以下の技術で構築されています：

  • フロントエンド： HTML5、CSS3、JavaScript (ES6+)
  • フォント： Noto Sans JP（Google Fonts）
  • データ形式： JSON
  • ストレージ： localStorage（ブラウザローカル保存）
  • ホスティング： GitHub Pages

サーバーサイドのプログラムは使用しておらず、すべてクライアントサイドで動作します。そのため、インターネット接続があれば、どこからでもアクセスできます。
