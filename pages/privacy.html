プライバシーポリシー

最終更新日：2025年11月

はじめに

JLPT N2 読解練習サイト（以下「当サイト」）は、ユーザーのプライバシーを尊重し、保護することを約束します。

このプライバシーポリシーは、当サイトがどのようにユーザー情報を取り扱うかを説明するものです。

🔒 収集する情報

1. 個人情報

当サイトは、個人を特定できる情報を一切収集しません。

  • 氏名、メールアドレス、電話番号などの個人情報は収集しません
  • ユーザー登録やログイン機能はありません
  • Cookie を使用した個人の追跡は行いません

2. 学習データ

以下の学習データがあなたのブラウザにローカルに保存されます：

  • 解答履歴（どの問題を解いたか）
  • 正答・誤答の記録
  • 学習日時
  • カテゴリー別の成績
  • 連続学習日数

重要： これらのデータは、あなたのブラウザの localStorage に保存され、サーバーには一切送信されません。

3. 技術情報

当サイトは静的サイトとしてホスティングされているため、以下の情報も収集しません：

  • IPアドレス
  • ブラウザの種類やバージョン
  • アクセスログ
  • クリックやページ遷移の追跡

💾 データの保存方法

localStorage（ブラウザローカルストレージ）

あなたの学習データは、Web Storage API の localStorage を使用して、あなたのブラウザ内にのみ保存されます。

localStorage の特徴：

  • データはあなたのデバイス内にのみ存在します
  • インターネットを経由してどこかに送信されることはありません
  • 他のユーザーや当サイトの管理者がアクセスすることはできません
  • ブラウザのキャッシュをクリアすると削除されます
  • 有効期限はなく、削除しない限り永続的に保存されます

データのエクスポート・削除：

「進捗」ページから、いつでも以下のことができます：

  • エクスポート： あなたの学習データをJSONファイルとしてダウンロード
  • 削除： すべての学習データを完全に削除

🌐 外部サービスの使用

Google Fonts

当サイトは、フォントの表示のために Google Fonts を使用しています。

  • Google Fonts の使用により、あなたのIPアドレスが Google に送信される可能性があります
  • Google のプライバシーポリシーは こちら をご確認ください

GitHub Pages

当サイトは GitHub Pages でホスティングされています。

  • GitHub のサーバーログには、アクセス情報が記録される場合があります
  • GitHub のプライバシーポリシーは こちら をご確認ください

アナリティクスツール

当サイトは、Google Analytics などのアクセス解析ツールを使用していません。

訪問者数やページビューなどの統計情報も収集していません。

🍪 Cookie の使用

当サイトは Cookie を使用していません。

ただし、Google Fonts などの外部サービスが Cookie を設定する可能性があります。

👶 子どものプライバシー

当サイトは、すべての年齢の学習者に適しています。個人情報を収集しないため、子どもが使用する場合でも特別な懸念はありません。

ただし、13歳未満のお子様がインターネットを使用する場合は、保護者の監督をお勧めします。

🔐 セキュリティ

当サイトは HTTPS を使用して暗号化された通信を提供しています。

ただし、学習データはあなたのブラウザに保存されるため、デバイスのセキュリティはあなた自身の責任となります。

セキュリティのヒント：

  • 共有デバイスを使用する場合は、使用後にブラウザのデータを削除することを検討してください
  • 公共のコンピューターでは、「進捗」ページからデータを手動で削除することをお勧めします
  • 重要なデータは定期的にエクスポートして、バックアップを取ることをお勧めします

🌍 データの国際転送

当サイトは個人データを収集しないため、データの国際転送は発生しません。

GitHub Pages のサーバーは米国に所在していますが、あなたの学習データがサーバーに送信されることはありません。

⚖️ ユーザーの権利

あなたは、あなたのデータに関して以下の権利を持っています：

  • アクセス： ブラウザの開発者ツールで localStorage の内容を確認できます
  • エクスポート： 「進捗」ページからデータをダウンロードできます
  • 削除： 「進捗」ページから、またはブラウザの設定ですべてのデータを削除できます
  • 拒否： localStorage を無効にすることで、データの保存を拒否できます（ただし、進捗管理機能が使えなくなります）

📝 プライバシーポリシーの変更

当サイトは、必要に応じてこのプライバシーポリシーを更新する場合があります。

変更があった場合は、このページに新しいポリシーを掲載し、「最終更新日」を更新します。

重要な変更がある場合は、サイトのトップページでお知らせします。

📧 お問い合わせ

このプライバシーポリシーに関してご質問がある場合は、GitHub の Issues ページからお問い合わせください。

📌 まとめ

要約：

  • ✅ 個人情報は一切収集しません
  • ✅ 学習データはあなたのブラウザにのみ保存されます
  • ✅ データはサーバーに送信されません
  • ✅ Cookie による追跡はありません
  • ✅ アクセス解析ツールは使用していません
  • ✅ あなたはいつでもデータを削除できます

当サイトは、あなたのプライバシーを最大限尊重しています。
